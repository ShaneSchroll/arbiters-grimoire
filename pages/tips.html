15 tips for asking better questions

How to talk to The Oracle.

The bot is grounded in the official Comprehensive Rules and can pull live oracle text from Scryfall. These notes help you frame questions so the answer cites the right rule, the right card, and the right corner of the turn structure.

15 tips
01/15
Fundamentals

Capitalize card names

When a card name is capitalized, the Oracle runs a Scryfall lookup so the ruling uses the card's exact oracle text.

Lowercase names get treated as plain English. Capitalize them and the bot fetches live oracle text — including any errata you might not have memorized.

how does animate land interact with humility?
I can answer in general, but if you mean Animate Land and Humility, capitalize the names and I'll pull oracle text for a more precise ruling.

Name the format

Commander, Modern, Legacy, Pauper, and Vintage have different banlists, mulligan rules, and starting life totals.

05/15
Precision

Invoke the layer system by name

For continuous effects, say “layer 6” or “type-changing effect” so the bot follows CR 613 directly.

Continuous effects apply in seven layers. If you suspect layers are the source of a weird interaction, mentioning the layer focuses the explanation on CR 613.

Order of operations for Blood Moon vs. Urborg, Tomb of Yawgmoth — which layer?
Both are type-changing effects in layer 4. Within a layer we use timestamp order, so whichever entered the battlefield later wins.

Ask one interaction at a time

Two questions in one prompt make for a fuzzier answer. Send them separately for cleaner rule citations.

The Oracle cites rules per interaction. Splitting a complex turn into multiple prompts gives you a clean ruling per question — and lets you correct course mid-thread.

Can I respond to a triggered ability with another triggered ability, and also, how does priority pass after combat damage?

Fundamentals

Paste the exact text if you're unsure of the name

Quote the ability text verbatim and the bot can identify the card or answer without needing the name at all.

If you have the physical card but you're blanking on the name, paste the rules text in quotes. The bot will either name it or work directly from the text.

What's the card that says “Whenever a player casts a creature spell, you may pay 1. If you do, copy that spell.”
That templating is closest to Fork-style copying on a permanent. Did you mean a creature, or a triggered-ability artifact? I can search either way.
09/15
Precision

Specify the zone

Battlefield, graveyard, exile, library, stack, hand, command — the zone changes which abilities can trigger.

Abilities only function in the zones their templating implies. A trigger that says “when this dies” only works while moving from battlefield to graveyard.

If my Bloodghast is in exile when a land enters, does it come back?
No. Bloodghast's landfall ability returns it from the graveyard, not exile. While in exile it has no abilities relevant to the trigger.
10/15
Structure

Say who controls what

Many effects only affect creatures “you control” or “an opponent controls.” Spelling this out prevents ambiguity.

Especially in multiplayer Commander, “my Anointed Procession” and “the opponent's Anointed Procession” give very different answers.

I control Anointed Procession. My opponent controls Parallel Lives. I create a token. How many do I get?
Only your Anointed Procession applies to tokens you create. Parallel Lives applies to your opponent's tokens. You'd create 2.

Replacement vs. triggered effects

When ordering matters, say which kind of effect is in play. Replacement effects modify events; triggered effects use the stack.

Replacement effects (“instead,” “if… would”) never use the stack and can't be responded to. Triggered effects do. Mention which you're asking about.

14/15
Fundamentals

Reference priority windows for stack questions

“Does my opponent get priority?” “Can I respond?” “What happens after this resolves?” — all priority questions.

Priority windows are the bone-structure of every stack interaction. Frame the question in priority terms and the answer becomes a clean step list.

I cast Lightning Bolt. After it resolves, does my opponent get priority before I draw for turn?
Yes. After any spell or ability resolves, the active player receives priority. You'd have priority first, then it passes around the table before the next phase or step begins.
